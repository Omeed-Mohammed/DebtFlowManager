# DebtFlowManager

## Overview
DebtFlowManager is a WinForms-based financial management system designed to manage suppliers, debts, and payments with a structured and scalable architecture.

## Architecture
- Layered Architecture (PL / BL / DAL / DTO)
- Separation of Concerns
- Repository Pattern (Data Access)

## Technologies
- C#
- .NET WinForms
- ADO.NET
- SQL Server
- Stored Procedures
- DTO Pattern

## Key Features
- Supplier Management
- Debt Recording
- Payment Tracking
- Financial Immutability Logic
- Soft Delete (IsActive)
- Audit Fields (CreatedBy / UpdatedBy)
- TRY/CATCH + Transactions
